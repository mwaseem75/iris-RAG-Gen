# Summary
Iris-RAG-Gen is a generative AI application that leverages the functionality of IRIS Vector Search to personalize ChatGPT with the help of the Streamlit web framework, LangChain, and OpenAI. The application uses IRIS as a vector store.

# Features
* Ingest Vector Data in IRIS
* Personalize ChatGPT
* OpenAI ChatGPT

# Installation
1. Clone/git pull the repo into any local directory

```
git clone https://github.com/mwaseem75/iris-RAG-Gen.git
```

2. Open a Docker terminal in this directory and run:

```
docker-compose build
```

3. Run the IRIS container:

```
docker-compose up -d 
```

To run the application Navigate to [**http://localhost:8051**](http://localhost:8051)
